Forecast Weather plugin for wordpress using Wunderground API.

== Description ==
Plugin for wordpress to forecast weather anywhere on your site using a shotcode [fw].  The site connects to Wunderground's JSON API.  You can display as many days with the attribute days.  The plugin displays relevant icons using a font instead of an image.  Font icon credit goes to Erik Flowers:
https://erikflowers.github.io/weather-icons/

Plugin works with any theme, no additional styling required.

== Installation ==
1.  Download Zip
2.  Upload plugin folder to uploads directory or with plugin uploader
3.  Activate plugin
